=================================================================================
1. 오브젝트 배치하기..
=================================================================================
	
	-------------------
	1.	프로젝트 생성..
	-------------------

		1]	[ 템플릿 선택 ]
			> [ 기본 ]

		2]	[ 프로젝트 세팅 ]
			>	[ 블루 프린트 ]
				[ 데스크톱 / 콘솔 ]
				[ 시작용 콘텐츠 포함 ]

		3]	[ 메뉴 ]
			> [ 파일 ]
			> [ 새 레벨.. ]
			> [ Basic ] 선택
		
			**	개발 팁..

				-	언리얼의 경우
					시스템 리소스를 많이 사용..

					-	그래픽 품질을 떨어뜨려
						연산 부담을 줄일 수 있음..
				
					-	[ 툴바 ]
						> [ Settings ]
						> [ Engine Scalability Settings ]
						> [ Medium ]

		6]	[ 메뉴 ]
			> [ 파일 ]
			> [ 현재 레벨을 다른 이름으로 저장 ] or 
			Ctrl + S 로 저장.
			-	적당한 이름으로 저장..
			-	"MapTest"


		7]	폴더 추가..
		
			-	[ 콘텐츠 브라우저 ]
				> [ Add ]
				> [ Folder ]
				> [ New Folder ]
				-	"Maps"로 변경..

			-	[ 마우스 우클릭 ]
				> [ 팝업 ]
				> [ Folder ]
				> [ New Folder ] 클릭..
				-	"Maps"로 변경..

		8]	6]의 레벨 파일을
			[ Maps 폴더 ]로 이동..
			

		 
	-------------------
	2.	액터 배치하기..
	-------------------

		-	액터..
			-	레벨에 배치할
				오브젝트..

		1]	[ 컨텐트 브라우저 ]
			> [ 검색창 ]
			> "Shape_QuadPyramid"
			검색 후 복제..
			-	[ Ctrl + D ]
			-	이름..
				-	"Shape_QuadPyramid_Clone"

		2]	[ 콘텐츠 브라우저 ]에
			[ Models ] 폴더 생성 후
			1]을 이동..

		3]	복제된 [ Shape_QuadPyramid_Clone ]을
			[ 뷰포트 창 ]으로
			드래그 앤 드롭.


	-------------------
	3.	머터리얼 적용..
	-------------------

		-	머터리얼..

			-	정점, 이미지, 광원등의
				정보를 연산하여 랜더링 방식을				
				설정하는 에셋..

		1]	[ 스태틱 메시 에디터 ] 실행..
			-	"Shape_QuadPyramid_Clone" 더블 클릭..
			-	[ 스태틱 메시 에디터 ]는
				"4. 스태틱 메시 에디터"에서 추가 설명..

		2]	재질 설정..
			-	[ 디테일 탭 ]
				> [ 머터리얼 슬롯 ]에서
				선택 가능..
				-	[ M_Tech_Hex_Tile_Pulse ] 선택..

		3]	저장..

		4]	[ 뷰포트 창 ] 확인..




	-----------------------
	4. 스태틱 메시 에디터..
	-----------------------

		----------
		1)	메뉴..
		----------

			------------
			-	콜리전..
			------------

				-	스태틱 메시 액터에
					콜리전을 설정..
					( 충돌 판정 )

				---------------
				-	주요 메뉴..
				---------------
				
					-----------------
					-	각종 단순화된
						콜리전 추가..
					-----------------

						-	구체, 캡슐, 박스..
						-	단순한 모양의 콜리전..
						-	모양이 단순하여
							연산에 부담이 적음..


					-----------------------
					-	10면체 X/Y/Z 단순화
						콜리전 추가..
					-----------------------

						-	모델 데이터를 10면체로
							인식하여 콜리전 추가..

						-	X/Y/Z는 콜리전 생성시
							기준 축..


					-----------------------
					-	18 / 26 면체 단순화
						콜리전 추가..
					-----------------------

						-	면체 수 증가..
							->	충돌관련 연산량 증가..


					-----------------
					-	콜리전 제거..
					-----------------

						-	설정된 콜리전 제거..


					------------------------
					-	자동 컨벡스 콜리전..
					------------------------

						-	메시 형태에 따라
							콜리전을 자동으로 생성..

						-	항목 선택시 [ 컨벡스 분해 ] 창이
							팝업 됨..
							-	생성할 콜리전의 정밀도 설정..

						-	단순화된 콜리전에 비해
							모양이 복잡..
							-	충돌관련 연산 부담 증가..

						-----------------------
						-	[ 컨벡스 분해 ]창..
						-----------------------

							-	[ 자동 컨벡스 콜리전 ] 선택..
											or
								[ 메뉴 ]
								> [ 창 ]
								> [ 컨벡스 분해 ]


							---------------
							-	주요 속성..
							---------------

								-	[ 헐 정밀도 ]
									-	콜리전 생성 시
										정밀도..

								-	[ 헐 개수 ]
									/ [ 최대 헐 버텍스 ]

									-	콜리전 생성시
										버텍스 개수..

									-	이 값을 증가시키면
										콜리전과 모델의 모양이
										점점 같아짐..

										-	충돌관련 연산량이 증가..
										-	정밀한 충돌이 필요하지 않다면
											초기 값 권장..
					
			--------
			-	창..
			--------

				-	스태틱 메시 에디터에서 다루는
					각 창의 표시 / 숨김 설정..


		------------
		2)	뷰포트..
		------------
			-	에셋의 상태를
				시각적으로 확인..

		---------------
		3)	디테일 탭..
		---------------
			-	해당 메시의 주요 속성 확인..
